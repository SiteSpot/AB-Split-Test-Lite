A/B testing plugin for WordPress. Test different variations of your content to improve conversions.

## Description

AB Split Test Lite allows you to run A/B split tests on any WordPress post, page, or custom post type. Test different variations of your content to find out which version converts best.

**Features:**

* Visual split test builder
* Works with Elementor, Beaver Builder, Bricks, Oxygen, and more
* Statistical confidence reporting
* Conversion tracking
* Goal tracking

## Installation

1. Upload the plugin files to the `/wp-content/plugins/ab-split-test-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> AB Split Test screen to configure the plugin.

## Frequently Asked Questions

### How do I create a test?

Navigate to any post or page and you will see the AB Split Test meta box where you can create variations.

### Does this work with page builders?

Yes, AB Split Test works with most popular page builders including Elementor, Beaver Builder, Oxygen, and more.
